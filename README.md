# Catamine

Cat image browser powered by [CATAAS](https://cataas.com) (Cat as a Service).

## Features
- Browse cats by tag
- Random cat images
- Generate cat images with custom text overlay
- Dark theme with amber accents

## Tech Stack
- Vanilla JS
- Tailwind CSS v4 (browser build)
- CATAAS API

## Usage if cloning locally
Open `index.html` in a browser. No build step required.
